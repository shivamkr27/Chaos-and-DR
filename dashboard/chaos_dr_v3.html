What This Project Is

Chaos Engineering + Disaster Recovery on AWS. The app runs on Primary (us-east-1). On failure, a Cloudflare Worker automatically routes all traffic to DR (us-west-2) — replacing Route53 (free tier friendly). Chaos experiments run intentional failures to validate recovery within defined SLO targets.

Stack

Node.js + Express → Docker → K3s (Kubernetes) → EC2 t3.micro → AWS
PostgreSQL RDS (primary) + Read Replica (DR)
Prometheus (metrics) · Cloudflare Worker (failover proxy)
